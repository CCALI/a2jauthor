Answers.prototype.getValue  

function(varName, varIndex)

Parameters

  1. varName {String}

    Name of the interview variable (not case-sensitive)

  2. varIndex {Number}

    The index used to access a specific value of a repeating variable. (optional)

    Returns the effective variable value:

    • If the variable is repeating and no index is requested, then an English comma delimited string is returned.

    • If the variable is NOT repeating or only contains a single value then an index of 1 is defaulted and that element is returned.
