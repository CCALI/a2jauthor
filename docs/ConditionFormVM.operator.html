ConditionFormVM.operator

  • property

{String}

 

This is the operator used in the expression that will determine whether the template should be rendered or not.

String

Possible values are: 'is-true', 'is-false', 'is-equal', 'is-not-equal', 'is-greater-than', and 'is-less-than'.
