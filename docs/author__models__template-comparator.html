author/models/template-comparator

  • module

{Object}

 

Provides functions that given a key and direction (asc or desc) will generate a comparator function meant to be used to sort Template.List instances through the sort plugin; the list will be sorted by the provided key and when templates have the same value for the given key, active templates will be sorted first.
