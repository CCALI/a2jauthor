<test-assemble-options>

  • module
author/templates/assemble/

{Module}

 

This component is used when user clicks the "Test Assemble" button, it exposes two buttons that allow the author to load an anwers file and/or get a PDF generated out of a single template or of all the templates matching the provided [guideId].

Module

Use

  <test-assemble-options {guide-id}="guideId" />
  <test-assemble-options {guide-id}="guideId" {template-id}="templateId" />
