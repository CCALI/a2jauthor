Avatars

  • stylesheet
avatars.less  

Avatars are used to represent both the guide and the client throughout A2J interviews in the Viewer application.

Avatars are added programmatically using javascript so that are displayed inline, rather than loaded in using <img> or <object> tags. Being loaded in inline makes them easier to style, but was a necessity to support Edge and IE11, where we were experiencing an issue loading them as
