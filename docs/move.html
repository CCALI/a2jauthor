move

  • function
 

Moves the item at the from index to the to index, it does not mutate the provided array, it returns a new reference.

Usage:

let a = [1, 2, 3];
let b = move(a, 0, 2);

assert.deepEqual(a, [1, 2, 3]);
assert.deepEqual(b, [2, 3, 1]);
