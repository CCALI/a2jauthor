loopCollection

  • property
repeatLoop.ViewModel.prototype.loopCollection

{can.List}

 

List of numbers progressing from 0 up to either loopCounter or the value of the variable reference by loopVariable. E.g, if loopVariable's value in answers is 5 the return list would be [0, 1, 2, 3, 4].
