Step Color Mixin

  • styles
step-color-mixin  

The .step-colors() mixin takes the color passed from the step colors array @steps and generates the CSS necessary for the left border color, hover, and active states for the Pages panel.

For example:

 #step15 {
  .step-colors(@step15);
 }
