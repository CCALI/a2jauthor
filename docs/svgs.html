Editing SVG Files

  • styles
svgs  

The SVG files can be edited in any vector graphics application (like Adobe Illustrator). However, because of the CSS reference and classnames required for A2J the files will also need to be opened in a text editor prior to usage. Once vector editing is completed and saved as an SVG, open the files in a text editor to ensure that the CSS references are in tact. Each file should have a reference to the CSS at the top of the file like:

<?xml-stylesheet href="../../styles/viewer-avatars.css" type="text/css"?>

Additionally, each file needs to have a classname set on the <svg> element of either avatar-guide or avatar-client along with a default skin tone classname like skin-darker. For example:

<svg version="1.1" id="Guide_-_Female_-_Front" class="avatar-guide skin-darker"
    xmlns="http://www.w3.org/2000/svg" xmlns:xlink="http://www.w3.org/1999/xlink"
   x="0px" y="0px" viewBox="0 0 214.4 757.6" style="enable-background:new 0 0 214.4 757.6;"
 xml:space="preserve">

Color Groups

Each SVG file also has several elements which contain unique IDs that determine what colors are applied to that portion of the vector image. For example, the avatar's hands will have id="left-hand" which allows the CSS to apply the skin tone to that part of the image. These IDs can be edited directly in Illustrator by creating groups and layers with these names or by adding them with a text editor.

Here is a list of currently used IDs that apply color to the SVGs:

  • Skin color applied to IDs: face, left-hand, right-hand, foot-right, foot-left
  • Variations on the selected skin tone to create gradients and shadows are applied to IDs: face-shadows, hand-shadow, and left-hand-graident_1_
  • Shirt or tie color is applied to ID shirt
  • Hair color is applied to IDs: hair, hair-front, hair-left, hair-right
  • A darkened variation on hair color used to create shadows is applied to IDs: hair-back, hair-top
