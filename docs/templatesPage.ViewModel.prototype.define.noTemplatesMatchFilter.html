noTemplatesMatchFilter

  • property
templatesPage.ViewModel.prototype.define.noTemplatesMatchFilter

{Boolean}

 

Whether the filter selected by the user has no matches, this property checks if the source list (templates) has items, if there is no searchToken and finally if the displayList is empty which means, that there are no matches for activeFilter.
