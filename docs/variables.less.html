Variables

  • stylesheet
variables.less  

Each CSS component has less variables that define basic values for styling that component. Each variable can be re-used anywhere in the component or app. For example, common colors, grid sizes, and margin / padding are all defined using variables.

The default Bootstrap variables.less file has been broken up into multiple files. Each CSS component file now contains it's own variables. This file @imports all the other variable files.

Alerts

alerts.less all colors for alert messages

Buttons

buttons.less for each of Bootstrap's buttons, define text, background and border color.

Close

close.less font settings for the close icon used in alert messages and modals

Colors

color-palette.less all the most common colors for the theme

Components

components.less Define common padding and border radius sizes and more. Values based on 14px text and 1.428 line-height (~20px to start).

Container sizes

container.less Define the maximum width of .container for different screen sizes.

Forms

forms.less color, padding, outline, margin, and border radius for all <input> types, as well as default colors for error and success states.

Grid system

grid.less Define your custom responsive grid. Currently 12 columns with 30px gutter width.

Labels

labels.less Background colors and text color for labels.

List Groups

list-group.less Colors, padding, border, hover effects, and link colors for list groups.

Media queries breakpoints

media-queries.less Define the breakpoints at which your layout will change, adapting to different screen sizes. Currently 480px, 768px, 992px, and 1200px.

Modals

modals.less Padding, color, header and footer settings for modals.

Navbars

navbar.less default settings for both horizontal and vertical navbars, including padding, width, height, and color.

Navs

navs.less defines the look of shared nav styles, tabs, pills, and toolbars.

Panels

panels.less variables for the color, padding, text, and border properties on panels.

Tables

tables.less Customizes the .table component with basic values, each used across all table variations.

Thumbnails

thumbnails.less padding and border for thumbnail images.

Typography

typography.less All the font sizes, headings, and weights for the theme

Z-Index

z-index.less Z-index master list.

Warning: Avoid customizing these values. They're used for a bird's eye view of components dependent on the z-axis and are designed to all work together.

Unused Variables

A2J Author does not use these Bootstrap components and variables-unused.less is not imported into the CSS. It's here as a placeholder for the required variables, in the event these components are supported in the future.
