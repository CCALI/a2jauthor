csh5/variables Variables Section

This section lists all variables added manually via the Add button or loaded from an existing interview.

Name

Variable names can contain letters, numbers, symbols and spaces. If spaces are used then use in the logic conditions requires using brackets around them.

Type The type determines how the data is sent to HotDocs. Also determines how calculations are performed.
Repeating If true, the variable is an array. [child#1] and [child#2]
Comment Optional author comment about the purpose of this variable

Options

Add Add a user defined variable. A type should be specified.

csh5/Steps Steps Section

Start/Exit points

Starting Point A2J will start the inteview on this question.
Exit Point A2J will save the answer file and switch to the next phase of data processing after this question.

Steps

Number Should be a 1 or 2 digit step number. If step is 0 then an arrow graphic is used instead.
Sign Text to display on the sign

Steps may be reordered using the position changer.

csh5/Pages Pages Section

Lists all pages in the interview. Popups are presented as stand alone pages at the bottom of the list. This allows a single popup to be used by multiple pages. Every page exists in one step. The page will appear under its step in alphabetical order. Select a page to clone or delete it. Or double-click to edit. To create a new page, select an existing page and press New.

csh5/Map Map Section

The map is provided as a tool for the author to visualize the flow of an interview and to ensure branches are handled correctly. It's possible to create questions first then link them together later. The map is a convenient way to identify some flow issues. Click and drag to reposition a question icon.

csh5/Files Files Section

This lists all files currently attached to the interview. (Technically, all files present in the guide's folder.) Files that are present but not used in the interview are flagged. Files that are missing (such as after an upload of an A2J 4 interview) are flagged.

csh5/AllLogic All Logic Section

All logic sections in the interview are displayed and editable here.

csh5/AllText All Text Section

All text blocks that are displayed to the user are listed and editable here. This includes question, help and button captions.

csh5/Preview Preview Section

Starts the A2J viewer.

csh5/Report Report Section

An analysis of all content in the interview appears here.

csh5/Publish Publish Section

This provides links to obtain publishable versions of an interview.

csh5/Interviews Interviews Section

This lists all interviews the author has created as well as sample interviews that are viewable.

Clone Select an interview to clone. You can use the clone to experiment freely.
Upload A2J 4 Guide Upload an A2J 4 interview. The interview must be in A2J 4 format. If a failure occurs, please open the interview in A2J 4, do a Save As, and upload that new version.
